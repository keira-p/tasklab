# 🧠 TaskLab — Taskmaster Forecasting

Understanding how Taskmaster series unfold using prediction and simulation.

## Overview

This project explores how Taskmaster outcomes can be:

- predicted using historical performance data
- simulated to estimate uncertainty
- tracked over time as a series progresses

## Key components

- **Prediction model**: estimates final outcomes from partial series data
- **Simulation engine**: runs many possible series to estimate win probabilities
- **Trajectory analysis**: shows how likelihoods evolve over time
- **Live framework**: updates predictions as new episodes are released

## Example insights

- Winners often emerge early, but not always
- Some series are decided quickly, others remain open until the end
- Simulation reveals uncertainty that point predictions hide
