TaskLab uses historical Taskmaster data to identify contestant archetypes and simulate future performance, estimating each contestant’s probability of winning as the series unfolds.

Based on A Ridiculously Comprehensive Taskmaster Database c/o Kaggle:

https://www.kaggle.com/datasets/sujaykapadnis/comedians-challenged-ridiculous-taskmaster-ds

Goals:
- use past data → learn patterns (archetypes)
- use current data → predict winner (simulation)
